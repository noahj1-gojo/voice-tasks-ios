# todos

Native iPhone App für Voice-Todos — gebaut mit Swift, SwiftUI, SwiftData und Mistral AI.

## Features
- Sprachaufnahme → Apple Speech Recognition → Mistral KI extrahiert Todos
- Eine flache Liste, offen oben und Erledigt-Sektion unten
- Dark/Light Mode, Apple HIG Design
- SwiftData Persistenz

## Setup
1. Projekt in Xcode öffnen: `Todos.xcodeproj`
2. Signing & Capabilities → eigenes Team auswählen
3. Einstellungen (⚙️) → Mistral API Key eintragen
4. ⌘R → auf iPhone deployen

## Requirements
- iOS 17+
- Xcode 15+
- Mistral API Key (console.mistral.ai)
