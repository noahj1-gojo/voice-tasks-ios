# Voice Tasks iOS

Native iPhone App für Voice Tasks — gebaut mit Swift, SwiftUI, SwiftData und Mistral AI.

## Features
- Sprachaufnahme → Apple Speech Recognition → Mistral KI Kategorisierung
- 4 Kategorien: Todos, Termine, Ziele, Reminders
- 2×2 Grid, Detailansicht pro Kategorie
- Dark/Light Mode, Apple HIG Design
- SwiftData Persistenz

## Setup
1. Projekt in Xcode öffnen: `VoiceTasks.xcodeproj`
2. Signing & Capabilities → eigenes Team auswählen
3. Einstellungen (⚙️) → Mistral API Key eintragen
4. ⌘R → auf iPhone deployen

## Requirements
- iOS 17+
- Xcode 15+
- Mistral API Key (console.mistral.ai)
